# Customer Churn Prediction

## Problem
Predicting which telecom customers are likely to churn.

## Data
Customer demographics, services, and billing information.

## Approach
- Cleaned missing values
- Encoded categorical features
- Trained Logistic Regression and Random Forest models

## Results
Random Forest achieved higher ROC-AUC.

## Business Impact
Allows proactive retention campaigns for high-risk customers.

## Next Steps
Add real-time scoring API and threshold tuning.
