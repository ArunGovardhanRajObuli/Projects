# Data Cleaning and Analysis of SuperStoreOrders dataset.

## Problem
Analyzing the data to understand the sales performance and trends.

## Data
Superstore orders dataset containing orders, customers, products, sales, profit, region, order date, shipping date and country columns.

## Approach we took
- Cleaned the missing values and duplicates.
- Converted dates to a standard datetime format.
- Created profit margin feature.
- Performed time-series and category analysis.

## Insights
- Identified the top 10 products based on sales.
- Found monthly sales trends.
- Compared the performance of all the regions. 

## Next Steps
- Building a forecasting model.
- Segmenting customers.
