# Sales Analysis & Business Intelligence Projects

This folder contains projects focused on data cleaning, exploratory analysis, and business insights using Python and SQL.

These projects simulate the work of a Data Analyst in a business environment.

---

## Projects

### 1. Sales Data Cleaning & Analysis
**Folder:** `Sales Data Cleaning & Analysis/`  

- Cleaned raw sales data (types, missing values, duplicates)
- Engineered profit margin feature
- Analyzed trends by time, region, and product
- Visualized monthly sales and category performance

**Skills:** Python, pandas, data cleaning, EDA, visualization

---

### 2. SQL Business Analysis
**Folder:** `SQL Business Analysis/`  

- Loaded cleaned data into SQLite
- Answered 10 business questions using SQL
- Analyzed customers, revenue, profitability, and growth

**Skills:** SQL, SQLite, business analysis, aggregation

---

## Tech Used

- Python (pandas, numpy, matplotlib, seaborn)
- SQL (SQLite)
- Jupyter Notebook

---

##  Recommended Reading Order

1. `Sales Data Cleaning & Analysis/`
2. `SQL Business Analysis/`

The second project builds on the cleaned data produced in the first.

---

## Purpose

These projects were built as part of a structured learning path to transition into a Data Analyst / Data Scientist role.
